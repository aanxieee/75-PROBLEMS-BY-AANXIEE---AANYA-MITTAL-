## How Each Project Is Structured

```
LVX_PROBLEM_N_PROJECTNAME/
├── app.py              ← Streamlit app
├── core/               ← Algorithm / model logic
├── data/               ← Sample data or JSON store
├── requirements.txt    ← Isolated dependencies
└── README.md           ← Problem → Solution writeup
```

Each project has its **own virtual environment** — no dependency conflicts.
